# Sebelas
Website SMKN 11 Bandung

## Installation
Follow each command below so that the installation process runs correctly.
### Update Composer
`````
composer update
`````
### Creating Database
`````
CREATE DATABASE sebelas;
`````
### Running Migrations
`````
php artisan migrate
`````
### Running Seeders
`````
php artisan db:seed
`````
